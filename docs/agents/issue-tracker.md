# Issue Tracker: GitHub

Issues for this repo live in GitHub Issues for `ericclemmons/opencode-threads`.

## Commands

Use the GitHub CLI (`gh`) for issue operations.

Examples:

```bash
gh issue list --repo ericclemmons/opencode-threads
gh issue view 123 --repo ericclemmons/opencode-threads
gh issue create --repo ericclemmons/opencode-threads --title "Title" --body-file issue.md
```

## Publishing Issues

When a skill says to publish an issue, create a GitHub Issue in `ericclemmons/opencode-threads`.

Apply the appropriate triage label from `docs/agents/triage-labels.md`.
